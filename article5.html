1.block与inline，一个大矩形里面从左到右三个小矩形，那么大矩形就是block元素，小矩形就是inline元素。div是一个标准的block元素， 一个块级元素会新开始一行并且尽可能撑满容器。而span是一个标准的行内元素，一个inline元素可以在 段落当中被包裹一些文字而不打乱段落的布局。

常见的块级元素有 div, form, table, p, pre, h1, dl, ol, ul 等。

常见的内联元素有 span, a, strong, em,label, input, select, textarea, img 等

2.max-width.设置block元素的width可以防止它从左到右撑满整个容器。使用 max-width 替代 width 可以使浏览器更好地处理小窗口的情况。

3.positon属性，任意 position: static; 的元素不会被特殊的定位。relative 表现的和 static 一样，除非你添加了一些额外的属性， 其他的元素的位置则不会受该元素的影响发生位置改变来弥补它偏离后剩下的空隙。fixed不能常用，对性能不好。absolute是相对于最近的 “positioned”祖先元素。

4.浮动，浮动元素可以向左或向右移动，直到它的外边缘碰到包含框或另一个浮动框的边框为止，之后的元素将围绕它。若section前面的div元素 有float:left属性，section会与其重叠，此时用。after-box{clear:left}可解决。

5.媒体查询是做Resposive Design网站所需要的强大工具。1. 2.@media screen and (max-width:600px) {nav li{ display：inline； } }

6.inline-block属性对象呈现为inline对象，内容作为block对象呈现。之后的inline元素会被排列在同一行内。比如我们可以给一个link（a元素） 值，inline-block属性使其既具有block的宽度高度特性又具有inline的同行特性。
