Docker基础
发布于:2021-3-28|阅读：18|标签：#Linux #容器

Docker 是基于 Linux kernel 的虚拟化工具，仅需要极低的系统资源使用就提供了 强大的虚拟化、资源隔离能力。易于复制分享的优点，能够保证开发与部署环境的一致性。

Docker 的基本结构包括：

Docker 客户端和服务器（C/S）：

Image： Image（镜像）

Registry： 默认的 Registry 是 Docker 公司提供的 Docker Hub， 你可以将 Docker Hub 类比于 GitHub。

Container： 它通过操作相应的镜像提供一个执行环境。

Docker使用基本流程：

  1. 查看镜像： docker images
  2. 创建容器运行镜像： docker container run [镜像名]
  3. 以虚拟terminal的形式并添加交互式效果运行镜像： docker container run -it ubuntu bash
  4. 终止运行的容器： docker container kill [containID]

常用命令：

  • 查找镜像：docker search httpd
  • 获取一个新的镜像：docker pull ubuntu:13.10
  • 删除镜像: docker rmi hello-world
